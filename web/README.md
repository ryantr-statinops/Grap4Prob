# 🎲 TTK-AI | Visualization of Law of Large Numbers (v2.0)

## Mục tiêu
Xây dựng web app cao cấp minh họa Luật số lớn bằng mô phỏng xúc xắc và đồng xu với giao diện hiện đại.

## Chức năng mới
- **Đa dạng mô phỏng**: Hỗ trợ cả Xúc xắc (6 mặt) và Đồng xu (2 mặt).
- **Giao diện Premium**: Thiết kế Glassmorphism, Dark mode, Responsive.
- **Hiệu ứng Animation**: Chuyển cảnh mượt mà, hiệu ứng "rolling/flipping".
- **Phân tích AI thông minh**: Đưa ra nhận định dựa trên dữ liệu thực tế.

## Công nghệ
- **Cấu trúc Modular**: Sử dụng ES Modules (JS).
- **Vanilla CSS**: Hệ thống Design System hiện đại.
- **Chart.js**: Trực quan hóa dữ liệu thời gian thực.

## Cách chạy
Mở file `index.html` bằng trình duyệt (Yêu cầu trình duyệt hiện đại hỗ trợ ES Modules).
